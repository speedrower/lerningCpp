# Learning C++

This repository was created to track my progress in learning C++. You will find various projects, exercises, and experiments that helped me understand more advanced C++ concepts.

## Repository Structure

- `01_control_flow/` – conditional statements, loops
- `02_functions/` – functions, overloading, recursion
- `03_memory/` – pointers, references, dynamic memory allocation
- `04_oop/` – object-oriented programming, classes, inheritance, polymorphism
- `05_stl/` – Standard Template Library: vectors, maps, sets, iterators
